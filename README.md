# web-page-project

## Project Overview
This is an individual University Web Development Project. It is an e-commerce website for a university demonstrating PHP,MySQL, and server side development skills.

## Features
- Product browsing and shopping cart
- Database integration (PHP/MySQL)
- Responsive web design
- Secure session management
- Login and Signup page

## Technologies Used
- **Frontend:** HTML, CSS
- **Backend:** PHP
- **Database:** MySQL (via phpMyadmin)
- **Server:** Apache
- **Tools:** VS Code, Git
